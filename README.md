# Ziki's Simple CLI Blackjack


#### This is a simple CLI Blackjack game made in python
> [!WARNING]

## Installation
Clone the repository
```sh
git clone https://github.com/Zikithezikit/ZikiBlackjackCLI
```
Get into the folder
```sh
cd ./ZikiBlackjackCLI
```
Run the .py file 
```sh
python3 ./jack.py
```
